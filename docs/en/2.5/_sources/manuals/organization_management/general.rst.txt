============
Login method
============

Introduction
------------

| This document aims to explain how to log in to the Organization management page.

.. _organization_manager_login:

Login
------------

#. | When logging in to the Organization management page, make sure to have both the Login ID (Username or E-mail address) and password ready.

   .. figure:: /images/ja/manuals/platform/login/exastro-login_v2-4.png
      :alt: Login
      :width: 600px
      :align: left
      :class: with-border-thin

   .. list-table:: Item description
      :widths: 40 200
      :header-rows: 1
      :align: left

      * - Item name
        - Description
      * - | Username or mail address
        - | Username configured by system administrator or E-mail address.
          | e.g. admin or admin@example.com ※
      * - | Password
        - | Password configured by system administrator
          | e.g. password  ※

   .. tip::
      |  Both the Login ID (Username or E-mail address) and Password are the ones configured to the Organization administrator by the system admin when they created the Organization in :doc:`../platform_management/organization`. 

#. | Update the password and press the :guilabel:`Send` button.

   .. figure:: /images/ja/diagram/login2.png
      :alt: First-time login
      :width: 600px
      :align: left
      :class: with-border-thin

#. | Update the account information and press the :guilabel:`Edit` button.

   .. figure:: /images/ja/diagram/login3.png
      :alt: First-time login
      :width: 600px
      :align: left
      :class: with-border-thin

#. | After logging in, a workspace list similar to the picture below will be displayed.

   .. figure:: /images/ja/diagram/login_workspace.png
      :alt: Login
      :width: 600px
      :align: left
      :class: with-border-thin
