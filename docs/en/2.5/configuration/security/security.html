1. Security¶

1.1. Introduction¶

We recommend implementing seperate measures for system risks that the Exastro system cannot deal with alone.
This document contains information regarding the different security measure that the system administrator should consider.

1.2. Security measures¶

Install and implement the following security measures if needed.

警告

We recommend configuring the Exastro Platform authentication function's option parameters for Endpoint for service(EXTERNAL_URL) and Endpoint for System management(EXTERNAL_URL_MNG) seperately from each other to prevent unauthorized access.
For more information, see インストール.
