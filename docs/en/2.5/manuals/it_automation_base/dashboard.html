3. Main menu (DashBoard)¶

3.1. Introduction¶

This document explains the ITA Main menu (Dashboard).

3.2. Main menu¶

The main menu displays Widgets.
The contents of each widget and their placements can be customized.
The customized dashboard is saved on the system per user. The user will be able to use their own customized dashboard even when logging in from different environments.
  1. Page structure
    Widgets No.1-5 are displayed by default.
    Widgets No.6-9 are not displayed by default (Users can add them by pressing Edit Dashboard → Add Widget)

3.3. Edit/Operate Dashboard¶

3.3.1. Edit DashBoard¶

Press the "Edit DashBoard" button to move to the edit page.
The following actions are available when editing.
  • Add Widget
    See "Add Widget".
  • Edit, Delete Widget
    See "Edit, delete Widget".
  • Confirm edit
    Press the Confirm edit button to save the edited widget contents.
  • Reset
    Press the Reset button to reset the edited widget contents. This will return the widget to the default state.
  • Delete edit
    Press the Delete edit button to delete the edited contents. This will return the widget to before the contents were edited.

3.3.1.1. Add Widget¶

  1. Select  Add Widget to display the widget selection.
    Select the desired widget and press Apply to add the widget.

Tip

Blank
Hovering the cursor over or under an existing widget displays a ＋Blank button.
Pressing the ＋Blank button adds 1 Blank.
If there is not widget within the line, selecting the blank deletes 1 blank.

Tip

Reset menu
The user can move panels by dragging and droping them from the "Menu group" widget to the "Menu set"
