2. Accessing the Main menu¶

2.1. Introduction¶

This document explains how to access the main menu.

2.2. Accessing the main menu¶

Users can access the main menu from "Workspace list/details" which is displayed after logging in.
  1. Access the Main menu
    Pressing the list's icon displays the main menu.
    Pressing the Workspace details icon displays the main menu.

    Tip

    ※Workspaces which the user does not have workspace role (permission) to will not be displayed.
