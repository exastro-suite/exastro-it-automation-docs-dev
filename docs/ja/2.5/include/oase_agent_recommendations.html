推奨事項¶

専用ユーザーの払い出し¶

OASE Agent の利用には、適切なメニューに紐付けされたロールを持つユーザーを作成することを推奨します。
※ユーザーとロールの作成については、ユーザー を参照してください。
※ロールとメニューの紐付けについては、ロール・メニュー紐付管理 を参照してください。
