1. 概要¶

1.1. はじめに¶

本書では、Exastro システムをコンテナ運用管理ツールであるDockerおよびPodmanに構築した際の特徴について説明します。

1.2. 特徴¶

Exastro システムを簡単に利用を始めたい場合に適しています。
高い可用性やサービスレベルのシステム構築を実現したい場合は、 Kubernetes 版 の利用を推奨します。
